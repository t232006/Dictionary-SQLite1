Коротко о приложении.

Индивидуальный help представляет собой таблицу слов и переводов, которая заполняется вами с целью иметь быстрый доступ к словам, который вы бы хотели поскорее запомнить. Для облегчения этой задачи в приложении предусмотрен целый комплекс упражнений: здесь и тесты со словом – вариантами перевода и наоборот, и упражнение на сбор слова из букв, и задание на соответствие, карточки, а также игра верно-неверно. Также сформированный вами help можно отправить на телефон, чтобы всегда иметь доступ к вашим словам. 

help.

help представляет собой таблицу, состоящую из полей:

·        Слово. Для определенности различают понятие «слово» и «перевод». Под понятием «слово» следует понимать слово из «родного» языка.

·        Translation. Перевод слова на другой язык.

·        Дата. Дата добавления этого слова.

·        Оценка. Оценка может быть от 0 до 6. Оценка 6 говорит о том, что слово полностью освоено. При тестировании за каждый верный ответ добавляется по 1 баллу, а за каждый неверный ответ может вычитаться от 1 до 2 баллов, в зависимости от задания.

·        Тема. Можно выбрать тему, с которой ассоциируется слово.

·        Отметка. Ставится пользователем или автоматически по определенным критериям. Отмеченные слова участвуют в тесте.

·        Релевантность. Указывает на то, насколько это слово востребовано вами (например, сколько раз вы его искали).

Поиск.

Чтобы найти нужное слово или фразу в словаре, нужно ввести это выражение или его часть в поле «найти слово». Можно искать как слово, так и перевод. Найденное слово будет указано в словаре стрелочкой слева. Для перехода в поле можно воспользоваться сочетанием клавиш Ctrl+P. Также предусмотрен т.н. «глубокий поиск», когда будут найдены все вхождения фрагмента введенного текста. Для «глубокого поиска» нужно нажать на кнопку. Чтобы выйти из режима поиска нужно нажать на кнопку еще раз.

Сортировка по критериям и отображение параметров.

Для того чтобы отсортировать таблицу по любому полю, достаточно нажать на заголовок поля. При нажатии на заголовок поля «Оценка/Релевантность» появится меню,

в котором можно выбрать параметр отображения: релевантность или оценку. Также здесь можно выбрать вариант отображения данных:

в виде шкалы     

в виде числа       

в виде числа и шкалы 

Работа с таблицей.

По умолчанию разрешено только чтение (перемещение по записям). Для того чтобы получить возможность редактировать help, необходимо нажать на кнопку 6 (или нажать на комбинацию Shift+Ins).

 

 

 


После нажатия на кнопку 6 возможно удалить запись нажатием на кнопку 2 (или нажать на клавишу Del). При редактировании слова станут активными кнопки 4 и 3, которые означают «отменить» и «подтвердить» соответственно. Редактирование слова позволяет также изменить принадлежность к теме. 

Для того чтобы добавить слово, нужно нажать на кнопку 5 (или на клавишу Ins). После этого появится диалоговое окно, в котором можно добавить одно или несколько слов. Если необходимо добавить, отредактировать или удалить тему, необходимо нажать в этом диалоговом окне на надпись «работа с темами».

Для перемещения по записям таблицы можно использовать кнопки 1 или клавиши PageUp, PageDown. 

В таблице можно отображать слова, фразы или всё вместе. Для этого нужно поставить соответствующий маркер на панели «отображать». Если нужно отображать только отмеченные слова, то нужно поставить соответствующий флаг.

При желании можно увеличить релевантность слова путем нажатия на кнопку «+» или нажатием на кнопку , при этом индикатор рядом с кнопкой увеличится на 1.

ВЫДЕЛЕНИЕ СЛОВ.

Слова можно выделять вручную, отмечая их в поле «Отм.», или по специальным критериям:

·        с оценкой меньше 6;

·        с оценкой больше 0;

·        за определенные даты;

·        по темам.

Можно также выполнять сложные выделения. Операция «ИЛИ» означает, что будут выделяться слова, удовлетворяющие хотя бы одному из указанных вами критериев. Например, можно выделать все слова за определенные даты, а также по каким-либо темам. Для того чтобы выполнялась операция «ИЛИ», необходимо, чтобы флаг «отображать только отмеченные» не стоял.

Операция «И» означает, что будут выделяться слова, удовлетворяющие всем указанным вами критериям. Например, можно выделить слова за определенные даты, которые при этом относятся к каким-либо темам. Для того чтобы выполнялась операция «И», необходимо, чтобы флаг «отображать только отмеченные» стоял.

синхронизация.

Программа позволяет копировать слова из другой базы данных, хранящейся в другом месте. Для этого имеется вкладка «синхронизация». В левом окне вкладки представлена база данных, откуда слова нужно копировать в текущую базу данных (правое окно). Для этого нужно сначала выбрать хранения базы данных, нажав на кнопку  (или F10). После этого в левом окне появятся те слова, которые отсутствуют в текущей базе данных. Можно выбрать некоторые слова или же все сразу (Ctrl + A). Далее нужно нажать на кнопку  чтобы копировать выделенные слова.

Также help можно отправить на ваш удаленный диск Google Drive, чтобы затем этот help установить в своем мобильном приложении. Для этого нажмите на кнопку  и войдите в свой аккаунт Google в браузере (если вы в него не вошли). Для скачивания словаря нажмите на кнопку . При этом будет предложено выбрать нужный help из списка загруженных на ваш удаленный диск.

Весь ваш help можно сохранить в таблицу Excel (при условии, если на вашем компьютере установлено приложение MS Excel). Для этого нажмите на кнопку

упражнения для запоминания слов.

Упражнения помогают быстрее запомнить слова и фразы. Для того чтобы добавить слова в задания, их нужно выбрать, поставив галочку напротив слова, а также воспользоваться выбором по специальным критериям. Далее обязательно поставить флажок «отображать только отмеченные», поскольку в задания будут включены только отображаемые слова (однако, никто не мешает вам упражняться со всем словарем, не ставя эту галочку). В правом нижнем углу окна есть шкала потенциала. Потенциал – это некая величина, обозначающая, насколько вы «не знаете» отмеченные слова. Когда она станет равна 0, считается, что вы выучили выбранный набор слов. Вы можете выбрать для отображения количество (баллов, которые нужно наборать), шкалу или процент, а также оставить строку пустой.  

слово-перевод и перевод-слово

Тестовое задание, где нужно выбрать для каждого слова один из вариантов перевода.

    

собрать слово

В этом задании предлагается собрать слово из букв или фразу из слов.

    

соответствие

В этом задании нужно для каждого слова выбрать его перевод.

верно-неверно

В этом задании нужно за 20 секунд дать как можно правильных ответов: правильно ли для слова подобран перевод. Вероятность правильности ½.

карточки со словами

Здесь предлагаются карточки, в которых есть либо слово, либо перевод. Нужно отметить, знаете ли вы это слово или нет.

    

Дополнительно для каждого слова или фразы можно настроить, чтобы карточка с этим словом или фразой появлялась на экране через определенный промежуток времени, который можно настроить.

внешний вид

Программа дает возможность гибкой настройки ее внешнего вида.
